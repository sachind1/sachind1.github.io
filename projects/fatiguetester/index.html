Fatigue Test

Machine

The goal of this project was to create a machine capable of testing engineering materials to find their fatigue properties. The idea grew out of wanting to determine whether to anneal the Berkeley Formula Racing (BFR) chassis to make it more resistant to vibrational loads from the engine.

Objectives

Create a stress-life (SN) curve for our thin-wall welded 4130 steel. This maximum cycle count required for a single specimen creating this curve is 10^6, or a million cycles. The 4130 steel sample will be 0.5" wide and range from 0.028" to 0.095", covering the range of tubes thicknesses we use. the endurance limit.

4130 Steel Normalized SN Curve

The desired load is 2000lbf in tension and compression. As some tests require a large number of cycles, the speed of the machine should be fast enough to allow for timely completion of tests.

Additionally, different tests will require different applied loads, and a way to easily and quickly change the load is needed.
